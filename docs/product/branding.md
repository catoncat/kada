# 品牌与命名

## 品牌主名

- 英文：`Kada`
- 中文：`咔哒`
- 组合写法：`Kada 咔哒`

## 定位语

`可控的分镜执行系统`

## 对外一句话

Kada 咔哒把“分镜意图”转成可执行、可回放、可验收的出图流程。

## 文案约定

- UI 主标题使用 `Kada 咔哒`。
- 业务动作优先使用“方案/分镜方案”，减少“预案”表述。
- 提示词与任务说明中避免“广告感”语义，统一使用“影楼专业成片质感”。

## Icon 概念（当前方向）

- 基础形：圆角方形底。
- 主体：三个纵向递进的小画框，代表分镜序列。
- 识别点：左上红色锚点连接到三个画框，表达“约束绑定”。
- 结果态：底部画框可加极简对勾，表达“通过验收”。
